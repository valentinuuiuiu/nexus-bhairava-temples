## 🌟 What Is This?

**Nexus Bhairava Temples** is an open-source collection of **64 MCP (Model Context Protocol) servers** — one for every major API and service in the modern tech ecosystem.

Each "Temple" is a guardian that connects your AI agent to a real-world service:
- **You talk to your AI** → AI talks to Slack, GitHub, Stripe, AWS, and 61 others
- **No restrictions. No paywalls. No token prisons.**
- **Pure open-source. Built for everyone.**

This is the foundation of **NewZyon** — a Personal AI (PAI) system built as a **Knowledge Augmented Network (KAN)**.

## ⚡ Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/valentinuuiuiu/nexus-bhairava-temples.git
cd nexus-bhairava-temples
```

### 2. Install Dependencies
```bash
pip install fastmcp httpx python-dotenv
```

### 3. Configure Your Tokens
```bash
cp .env.example .env
# Edit .env with your API tokens
```

### 4. Start Any Temple
```bash
# Start Temple 3 — Slack
python slack_mcp_server.py

# Start Temple 27 — Anthropic/Claude
python anthropic_mcp_server.py

# Start Temple 64 — NewZyon Sacred Core
python newzyon_mcp_server.py
```

### 5. Connect to Claude Desktop
Add to your `claude_desktop_config.json`:
```json
{
  "mcpServers": {
    "slack-temple": {
      "command": "python",
      "args": ["/path/to/slack_mcp_server.py"]
    },
    "github-temple": {
      "command": "python",
      "args": ["/path/to/github_mcp_server.py"]
    }
  }
}
```

## 🧬 The Architecture

### The addon.py Pattern
Every temple follows the same sacred pattern:

```
service_addon.py          ← Pure API client class
service_mcp_server.py     ← FastMCP tools wrapper
```

**Why this pattern?**
- **Reusable** — addon works standalone or inside MCP
- **Testable** — test API calls without MCP overhead
- **Extendable** — add tools without touching the core
- **Consistent** — every temple feels the same

### Building a New Temple
```python
# 1. Copy the pattern
cp slack_addon.py myservice_addon.py
cp slack_mcp_server.py myservice_mcp_server.py

# 2. Modify the addon class
class MyServiceAddon:
    API_BASE = "https://api.myservice.com/v1"
    # ... your methods

# 3. Add MCP tools
@mcp.tool()
async def myservice_action(param: str) -> str:
    result = await client.get(f"/endpoint/{param}")
    return json.dumps(result)

# 4. Done — Temple 65 stands watch 🔱
```
